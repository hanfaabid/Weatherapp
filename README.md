# WeatherApp 🌤️

A **React Native** weather application that allows users to search for weather information for any city or select from a predefined list of cities. The app fetches live data from the **OpenWeatherMap API** and displays temperature, weather type, and other details like humidity, pressure, wind speed, and visibility.  

---

## Features

- Search for any city using the search bar  
- Horizontal scrolling list of predefined cities  
- Detailed weather information including:
  - Temperature (°C)  
  - Weather condition (Clear, Rain, Clouds, etc.)  
  - Pressure  
  - Humidity  
  - Wind speed  
  - Visibility  
- Beautiful UI with city images  
- Responsive design using `react-native-size-matters`
